## â“ FAQ

### Q: Apakah dark mode menghemat baterai?

**A:** Ya! Khususnya untuk device dengan OLED screen. Setiap pixel gelap = tidak ada backlighting.

### Q: Apakah tema saya tersimpan permanen?

**A:** Ya, tersimpan di browser cache. Tapi akan reset jika:

-   Anda clear browser cache/history
-   Anda menggunakan private/incognito mode
-   Anda menggunakan browser berbeda

### Q: Bisakah saya menggunakan dark mode hanya untuk satu halaman?

**A:** Tidak, tema berlaku global untuk seluruh aplikasi.

### Q: Apakah ada pilihan tema lain selain light dan dark?

**A:** Saat ini hanya ada 2 pilihan. Tapi bisa ditambahkan di masa depan!

### Q: Bagaimana jika tombol toggle tidak berfungsi?

**A:**

1. Refresh browser (F5)
2. Clear cache (Ctrl+Shift+Delete)
3. Coba browser lain
4. Hubungi support jika masalah berlanjut

### Q: Apakah dark mode bekerja di semua browser?

**A:** Ya! Chrome, Firefox, Safari, Edge - semua didukung.

## ğŸ”§ Troubleshooting

### Problem: Tombol toggle tidak terlihat

**Solution:**

-   Scroll navbar ke kanan (mungkin tertutup di mobile)
-   Refresh halaman
-   Clear browser cache

### Problem: Tema tidak tersimpan

**Solution:**

-   Pastikan browser support LocalStorage
-   Disable private/incognito mode
-   Check browser settings

### Problem: Text tidak readable di dark mode

**Solution:**

-   Refresh halaman
-   Clear browser cache
-   Cek brightness monitor

### Problem: Animasi terasa lambat

**Solution:**

-   Close tab lain yang berat
-   Disable browser extensions
-   Update browser ke versi terbaru
